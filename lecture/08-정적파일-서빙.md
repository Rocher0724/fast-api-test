# 8장. FastAPI에서 정적 파일(HTML/CSS/JS) 서빙하기

## 학습 목표

FastAPI 서버에서 바닐라 HTML/CSS/JS 파일을 **어떻게 제공하는지** 이해하고, 관련 에러를 해결하는 방법을 배운다.

---

## 8.1 정적 파일이란?

### 개념

서버가 **별도 처리 없이 그대로 전달**하는 파일을 정적 파일(Static Files)이라고 한다.

```
정적 파일:  HTML, CSS, JS, 이미지, 폰트 → 그대로 브라우저에 전달
동적 응답:  API 응답 → 서버가 데이터를 처리한 뒤 결과를 전달
```

### 왜 설정이 필요한가?

FastAPI는 기본적으로 **API 서버**다. HTML 파일을 자동으로 제공하지 않는다.

```python
# 이 코드만으로는 HTML 파일을 제공할 수 없다
app = FastAPI()

@app.get("/api/users")
def get_users():
    return [{"name": "홍길동"}]

# 브라우저에서 http://localhost:8000/index.html 접속 → 404!
```

## 8.2 정적 파일 설정 방법

### 프로젝트 구조

```
my-project/
├── app/
│   ├── __init__.py
│   └── main.py          # FastAPI 서버 코드
├── static/               # 정적 파일 폴더
│   ├── index.html
│   ├── style.css
│   └── script.js
└── requirements.txt
```

### FastAPI에서 정적 파일 서빙 설정

```python
from fastapi import FastAPI
from fastapi.staticfiles import StaticFiles
from fastapi.responses import FileResponse

app = FastAPI()

# 정적 파일 디렉토리 마운트
# "/static" URL 경로로 접근하면 "static" 폴더의 파일을 제공
app.mount("/static", StaticFiles(directory="static"), name="static")

# 루트 경로(/)로 접속하면 index.html을 제공
@app.get("/")
def read_root():
    return FileResponse("static/index.html")

# API 라우트
@app.get("/api/users")
def get_users():
    return [{"name": "홍길동"}]
```

### 접속 확인

```
http://localhost:8000/              → index.html이 보임
http://localhost:8000/static/style.css  → CSS 파일이 로드됨
http://localhost:8000/api/users     → JSON 응답
```

## 8.3 정적 파일 관련 에러들

### 에러는 어디서 확인하는가?

| 증상 | 확인 장소 |
|------|----------|
| 페이지가 안 나옴 (404) | 브라우저 주소창 + 서버 터미널 |
| CSS/JS가 안 먹음 | 브라우저 Network 탭 (파일 로딩 여부) |
| 이미지가 안 보임 | 브라우저 Network 탭 (파일 경로) |
| 페이지가 깨져서 나옴 | 브라우저 Console 탭 (JS 에러) |

### 404 - 파일을 못 찾는 경우

```python
# ❌ 디렉토리 경로가 틀림
app.mount("/static", StaticFiles(directory="statics"), name="static")
#                                             ↑ 오타: 'statics' 폴더가 없음

# ❌ 상대 경로 문제 (배포 시 자주 발생)
app.mount("/static", StaticFiles(directory="static"), name="static")
# uvicorn을 프로젝트 루트가 아닌 다른 경로에서 실행하면 못 찾음

# ✅ 절대 경로로 안전하게 설정
import os
BASE_DIR = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
STATIC_DIR = os.path.join(BASE_DIR, "static")

app.mount("/static", StaticFiles(directory=STATIC_DIR), name="static")
```

### CSS/JS가 로딩 안 되는 경우

```html
<!-- ❌ 경로가 잘못됨 -->
<link rel="stylesheet" href="style.css">
<!-- 상대 경로는 현재 페이지 URL 기준이라 예상과 다를 수 있음 -->

<!-- ✅ 절대 경로 사용 -->
<link rel="stylesheet" href="/static/style.css">
<script src="/static/script.js"></script>
```

**확인 방법:** 브라우저 Network 탭에서 CSS/JS 파일 요청의 **Status**가 200인지 404인지 확인한다.

### HTML에서 API 호출 시 주소 문제

```javascript
// ❌ 로컬에서 하드코딩하면 배포 시 안 됨
fetch("http://localhost:8000/api/users")

// ✅ 상대 경로로 쓰면 어디서든 동작
fetch("/api/users")
```

## 8.4 Templates (Jinja2) 방식

### 정적 파일과의 차이

정적 파일은 **그대로** 보내지만, 템플릿은 서버에서 **데이터를 넣어서** 보낸다.

```
정적 파일: HTML 그대로 전달 → 프론트에서 JS로 API 호출해서 데이터 표시
템플릿:    서버가 HTML에 데이터를 넣은 뒤 전달 → 완성된 HTML이 브라우저에 도착
```

### Jinja2 템플릿 설정

```bash
pip install jinja2
```

```python
from fastapi import FastAPI, Request
from fastapi.templating import Jinja2Templates

app = FastAPI()
templates = Jinja2Templates(directory="templates")

@app.get("/users")
def user_page(request: Request):
    users = [{"name": "홍길동"}, {"name": "김철수"}]
    return templates.TemplateResponse(
        "users.html",
        {"request": request, "users": users}
    )
```

```html
<!-- templates/users.html -->
<html>
<body>
    <h1>사용자 목록</h1>
    {% for user in users %}
        <p>{{ user.name }}</p>
    {% endfor %}
</body>
</html>
```

### 템플릿 에러 확인

```
# Jinja2가 설치 안 되어 있으면
ModuleNotFoundError: No module named 'jinja2'
→ 해결: pip install jinja2 + requirements.txt에 추가

# 템플릿 파일을 못 찾으면
jinja2.exceptions.TemplateNotFound: users.html
→ 해결: templates 폴더에 파일이 있는지, 폴더 경로가 맞는지 확인
```

## 8.5 바닐라 HTML + FastAPI 프로젝트 구조 가이드

### 추천 프로젝트 구조

```
my-project/
├── app/
│   ├── __init__.py
│   ├── main.py           # FastAPI 앱 + 라우트
│   ├── models.py          # DB 모델
│   └── database.py        # DB 연결 설정
├── static/                # 정적 파일
│   ├── css/
│   │   └── style.css
│   ├── js/
│   │   └── script.js
│   └── images/
│       └── logo.png
├── templates/             # HTML 템플릿 (Jinja2 사용 시)
│   ├── index.html
│   └── users.html
├── .env                   # 환경변수 (Git에 안 올림)
├── .gitignore
├── requirements.txt
└── runtime.txt            # Python 버전 지정 (Render용)
```

### main.py에서 모든 것을 연결

```python
from fastapi import FastAPI
from fastapi.staticfiles import StaticFiles
from fastapi.responses import FileResponse

app = FastAPI()

# 정적 파일 설정
app.mount("/static", StaticFiles(directory="static"), name="static")

# 메인 페이지
@app.get("/")
def home():
    return FileResponse("static/index.html")

# API 라우트들
@app.get("/api/users")
def get_users():
    return [{"name": "홍길동"}]
```

## 핵심 정리

1. FastAPI는 기본적으로 API 서버 — **StaticFiles 설정 없이는 HTML/CSS/JS를 제공하지 않는다**
2. `app.mount("/static", StaticFiles(directory="static"))` 으로 정적 파일 서빙
3. HTML에서 CSS/JS 경로는 **절대 경로** (`/static/style.css`) 사용이 안전
4. API 호출 시 `localhost:8000` 하드코딩 금지 — **상대 경로** (`/api/users`) 사용
5. 배포 시 파일 경로 문제가 자주 발생 → `os.path`로 절대 경로 설정
6. CSS/JS가 안 먹으면 **Network 탭**에서 파일이 실제로 로딩되는지 확인
